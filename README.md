## ⚙️ Running Locally

### Clone the repository

```bash
git clone https://github.com/AfrasyabZahid/afrasyab-portfolio.git
```

### Navigate into the project

```bash
cd afrasyab-portfolio
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
